# Color Finder (workspace folder)

The app files live in the **parent directory** (`colormatch/`), not in this folder.

## Run locally

From this folder:

```bash
npm start
```

Or from the repo root:

```bash
cd ..
python3 -m http.server 8080
```

Open **http://localhost:8080** (not `file://`).
